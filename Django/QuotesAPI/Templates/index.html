Introduction

The Quotes API is a powerful and versatile tool designed to provide developers with access to an extensive collection of inspirational, motivational, and thought-provoking quotes. This API allows users to seamlessly integrate inspiring quotes into their applications, websites, or projects, enriching content and user experiences. Whether you're building a motivational app, enhancing your blog, or looking to add a touch of wisdom to your project, the Quotes API simplifies the process of accessing and displaying quotes with minimal effort.

Usage

To retrieve a Quotes from the API, make a GET request to the API endpoint with the help of fetch or Ajax of JavaScript. The response from the server will contain a JSON object with the Quotes and Author Name.

You can retrieve the Quotes in JavaScript by Using the Fetch. Code Sample is Given Below: 

                    
       fetch("http://randomjokesapi.santosh0.com.np/JokesAPI/")
       .then(response=>response.json())
       .then(response=>{
           console.log(response[0]);
       }).catch(e=>{
           console.log(e);
       })

You can retrieve the Quotes in JavaScript by Using the Ajax. Code Sample is Given Below: 

                    
        let xhr = new XMLHttpRequest();
        xhr.open('GET', 'http://randomjokesapi.santosh0.com.np/JokesAPI/');
        xhr.onreadystatechange = function() {
        if (xhr.readyState === 4 && xhr.status === 200) {
            let data = JSON.parse(xhr.responseText);
            console.log(data[0]);
            }
        };
        xhr.send();
